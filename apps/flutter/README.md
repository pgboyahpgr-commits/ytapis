# ytapis &mdash; Flutter Desktop App

Native Windows GUI app for searching YouTube. Built with Flutter Desktop using the `ytapis` pub.dev package.

Built and managed by [geethudinoyt](https://github.com/geethudinoyt).

## Features

- Search YouTube with text queries
- Browse results in a scrollable list
- Tap any result to see details and play the video in your browser

## Run from source

```bash
flutter pub get
flutter run
```

## Build .exe

```bash
flutter build windows
```

Output: `build\windows\x64\runner\Release\ytapis_desktop.exe` (80 KB, requires adjacent DLLs)

## Download

Pre-built `.exe` available on the [Releases](https://github.com/pgboyahpgr-commits/ytapis/releases) page.
